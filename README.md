# Quick Event

#### 介绍
基于QT-5.6封装的一套控制与界面完全分离的代码；利用QT的元对象属性，实现控制类的自动实例化；VS-2015编译通过，使用了C++11语法；使用该模型可以更加专注与自己模块或功能点的开发，更便于调试和扩展；完全分离的控制和界面，让重构变得非常轻松；

#### 软件架构
1.  界面与控制逻辑完全分离
2.  事件管理
3.  线程管理
4.  控制类自动化管理
